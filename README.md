# machine_learning_basic

## 목적
본 스터디에서는 세 가지에 목표를 두고 있습니다.

1. 일반적인 지도학습 과정의 체계화
2. 보편적으로 널리 알려진 문제들에 대한 학습 및 해석
3. 빠르고 정확한 데이터 전처리

## 본 스터디를 통해서

- 지도학습 과정 체계화
